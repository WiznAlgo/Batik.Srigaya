# Batik Srigaya — Full Stack Editorial Website

Project ini adalah website premium bergaya editorial untuk Batik Srigaya.

## Tech Stack
- React
- Vite
- Tailwind CSS v4
- Express API

## Folder utama
- `index.html`
- `src/`
- `server/`
- `src/assets/` berisi gambar yang sudah dimasukkan

## Jalankan
```bash
npm install
npm run dev
```

Frontend berjalan di Vite dan API di Express.
Saat production:
```bash
npm run build
npm run server
```

## Catatan
- Form contact menyimpan data inquiry sederhana ke `server/data/inquiries.json`
- Semua gambar yang kamu kirim sudah dimasukkan ke dalam portfolio/lookbook
